## maven과 gradle 차이
    Build System의 일종. 소스 코드 -> 실행 가능한 어플리케이션 생성  
    (의존성 관리, 코드 패키징, 컴파일, 테스트 실행 등)
    maven에서 gradle로 점차 이행 중
**gradle의 장점**: 
가독성, 설정의 간편함, 다양한 언어 지원(java, kotlin, groovy),
빌드 스크립트 커스터마이즈 가능, 빌드와 테스트 속
